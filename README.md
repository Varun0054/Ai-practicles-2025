# AI Practicals 2025

This repository contains implementations of various AI algorithms and applications.

## Practicals List

1. **Depth First Search (DFS)** - `prac1.py`
   - Implementation of recursive DFS algorithm
   - Handles undirected graphs
   - Includes demo with sample graph

2. **Breadth First Search (BFS)** - `prac2.py`
   - Recursive BFS implementation
   - Level-order traversal approach
   - Full graph coverage with components

3. **A* Search Algorithm** - `prac3.py`
   - A* pathfinding implementation
   - Manhattan distance heuristic
   - Grid-based demo with obstacles

4. **N-Queens (Constraint Satisfaction)** - `prac4.py`
   - Branch and bound with backtracking
   - Solution counting optimization
   - Demonstrations for N=4 and N=8

5. **Selection Sort (Greedy)** - `prac5.py`
   - Greedy selection sort implementation
   - Step-by-step demonstration
   - Basic correctness assertions

6. **Minimum Spanning Tree (Kruskal's)** - `prac6.py`
   - Kruskal's MST algorithm
   - Union-Find data structure
   - Edge weight-based sorting

7. **Minimum Spanning Tree (Prim's)** - `prac7.py`
   - Prim's MST algorithm
   - Priority queue implementation
   - Connected component handling

8. **Kruskal's MST (Enhanced)** - `prac8.py`
   - Optimized Kruskal's implementation
   - Path compression in Union-Find
   - Detailed visualization

9. **Simple Chatbot** - `prac9.py`
   - Basic restaurant chatbot
   - Pattern matching responses
   - Common query handling

## Usage

Each practical can be run independently:

```powershell
python pracN.py  # where N is the practical number (1-9)
```

## Requirements

- Python 3.6+
- No external dependencies required

## Implementation Details

Each practical includes:
- Clear implementation with comments
- Demo function showing usage
- Basic error handling
- Assertions for correctness
